## Opciones de paquete

### Launch Site

Para negocios nuevos que necesitan presencia online clara y rápida.

- Un mensaje principal.
- Navegación bilingüe.
- CTA de contacto y analytics básicos.

### Conversion Site

Para marcas que quieren SEO, contenido y más generación de leads.

- Estructura multi-página.
- Base lista para blog.
- Mejor tracking de eventos.

### Web + Mobile Platform

Para equipos que necesitan algo más que un sitio vitrina.

- Lógica de producto personalizada.
- Flujos preparados para mobile.
- Arquitectura para crecer con el tiempo.
